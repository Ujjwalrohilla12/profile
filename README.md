## Portfolio Overview

This workspace is my personal profile site showcasing:

- Hero section with animated entrance, custom theming and personal branding.
- About, skills, education, and experience sections with structured layouts.
- Projects feed with live repo links and metadata from GitHub.
- Contact section with accessible forms and CTA flows.
- Dark/Light mode toggling and theme persistence.
- Reusable UI components built with Radix UI + Tailwind.

## Features

1. **Dark + Light Theme**: Uses class-based theme variables and `next-themes` for system preference fallback.
2. **Responsive navigation**: Mobile drawer toggles + desktop links.
3. **Motion design**: `framer-motion` scroll reveal, hover transitions, and fluid entrance animations.
4. **Semantic accessibility**: Proper heading order, ARIA labels, keyboard-friendly components, and color contrast.
5. **API integration**: GitHub repository list from REST API.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Project Structure

- `src/App.tsx` — app layout, router, providers.
- `src/pages/Index.tsx` — homepage assembly.
- `src/components/portfolio/` — section components.
- `src/components/ui/` — reusable UI primitives.
- `src/index.css` — CSS variables and theme tokens.
